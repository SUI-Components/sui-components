## Installation

```sh
$ npm install @s-ui/react-atom-help-text --save
```

## Usage

### Basic usage
```js
import AtomHelpText from '@s-ui/react-atom-help-text'

return (
  <div>
    <input style={input}
      type='text'
    />
    <AtomHelpText text='hello help text'/>
  </div>
)
```
