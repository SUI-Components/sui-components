## Installation

```sh
$ npm install @s-ui/react-atom-radio-button --save
```

## Usage

```js
import AtomRadioButton from '@s-ui/react-atom-radio-button'
```

### Basic usage

```js
<AtomRadioButton />
```

### Checked

```js
<AtomRadioButton checked />
```

### Disabled

```js
<AtomRadioButton disabled />
```

### w/ onChange handler

```js
<AtomRadioButton onChange={console.log} />
```
